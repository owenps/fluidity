# Tile Movement Shortcuts

Fluidity uses direct Commands for moving the focused Tile on the Workspace Grid: `tile.move.left`, `tile.move.down`, `tile.move.up`, and `tile.move.right`.

Default Keybinds: `⌥⇧` + H/J/K/L, with arrow-key aliases. This matches common tiling-window-manager muscle memory: `⌥` focuses, `⌥⇧` moves, and `⌃⌥` resizes.

H/J/K/L are the primary bindings because `⌥` + arrow keys commonly conflict with macOS/editor word navigation. Arrow aliases remain available for discoverability.
